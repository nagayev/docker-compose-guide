## Запуск 
Предполагается, что у вас уже установлены docker и docker-compose.  
Сборка:  
`sudo docker-compose build`  
Запуск:  
`sudo docker-compose up`  
Должно написать Docker-Compose is magic! в консоли и при заходе на localhost:1234
